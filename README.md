# CodeHud
A custom interface for tf2. This hud was made from ahud (https://github.com/n0kk/ahud) Credits to n0kk.
####Features

* Custom colors
* Custom crosshairs
* 6v6 scoreboard
* Alternative styles
* Home Server button
* 16:9, 16:10, and 4:3 support
* Competitive and MvM support
